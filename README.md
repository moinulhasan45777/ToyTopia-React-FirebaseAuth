Project Name: ToyTopia | A Local Kids Toy Store Platform

Purpose: It's purpose is to be a vibrant and playful online marketplace for kids' toys, encouraging families to discover and support local toy sellers.. Users can log in, browse toys, view detailed info, and leave feedback or ratings for toys they want to buy for their kids.

Live URL: https://toy-topia-d56cc.web.app/

Key Features: Users can Register, Sign-in, Sign-out, change their password if forgotten, Update their profile info including their image, View Toys and Buy Toys.

NPM Packages: tailwindcss, axios, lucide-react, react, react-dom, react-icons, react-router, react-toastify, swiper, vite, daisyui, firebase
